This project predicts whether a bank customer will churn (leave the bank) using Machine Learning.
🎯 Objective
To build a classification model that identifies customers likely to churn based on financial and personal details.
🛠 Tech Stack
Python
Pandas
Scikit-learn
⚙️ Process
Data Cleaning
Encoding Categorical Variables
Train-Test Split (80/20)
Logistic Regression Model
Model Evaluation
📊 Result
✅ Accuracy Achieved: 81.4%
🚀 How to Run
Bash
Copy code
pip install pandas scikit-learn
python churn.py
📌 Conclusion
This project demonstrates the practical application of Machine Learning in solving real-world business problems like customer retention and churn prediction.
